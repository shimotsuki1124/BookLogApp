# Booklog App

Flask を使った読書記録アプリです。

## 機能概要

- 書籍の読書中 / 読了状態の管理
- メモの記録
- ログ表示機能

## セットアップ手順

```bash
git clone https://github.com/yourname/booklog-app.git
cd booklog-app
python -m venv venv
source venv/bin/activate  # Windowsなら venv\Scripts\activate
pip install -r requirements.txt
python app.py
